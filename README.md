QR Code Component Project Challenge

A responsive QR Code Component built with HTML and Tailwind CSS as part of the Frontend Mentor challenge.

Overview

This project recreates the provided QR code card design using Tailwind CSS. The layout is mobile-first and adapts to larger screen sizes while maintaining a clean, centered design.

Features

- Responsive layout
- Mobile-first design
- Built entirely with Tailwind CSS
- Semantic HTML
- Centered card using Flexbox

Technologies Used

- HTML5
- Tailwind CSS
